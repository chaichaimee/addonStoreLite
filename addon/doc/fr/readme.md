# addonStoreLite

<p align="center">

---

## Description
**addonStoreLite** est une extension intelligente et légère qui transforme la façon dont NVDA annonce les éléments dans la boutique d'extensions. Elle supprime l'encombrement inutile — comme les étiquettes de statut, les adresses e-mail et les numéros de version superflus — pour ne présenter que les informations essentielles : **nom de l'extension, version, auteur, date d'installation et un indicateur clair « Désactivé » pour les extensions désactivées**. Le résultat ? Une annonce propre et concise qui vous permet de parcourir les extensions disponibles sans effort.

> **Fonctionne automatiquement :** Installez simplement l'extension et placez le focus sur n'importe quel élément de la liste dans la boutique d'extensions. NVDA prononcera immédiatement la version simplifiée. Aucune configuration requise.

<br>

## Pourquoi vous allez l'adorer
Parcourir la boutique d'extensions signifiait auparavant écouter de longues chaînes de texte répétitives — statut, canal, éditeur, etc. Avec **addonStoreLite**, vous n'entendez que ce qui compte, ce qui permet de gagner du temps et de réduire la charge cognitive. Elle s'adapte intelligemment au format classique avec étiquettes (NVDA 2025.x) ainsi qu'au nouveau format positionnel introduit dans NVDA 2026.x, garantissant des performances impeccables dans tous les onglets :

* ✅ Extensions installées
* ✅ Extensions disponibles
* ✅ Onglet mises à jour
* ✅ Canaux Beta & Dev

## Ce qu'il supprime
* Les étiquettes de statut comme « Statut : Activé » (seul « Désactivé » est annoncé si applicable)
* Les adresses e-mail entre chevrons
* Les noms de canaux (Externe, Stable, Beta, Dev)
* Les champs de version redondants (ex: « Version minimale de NVDA », « Dernier test »)
* La ponctuation et les espaces superflus pour un flux naturel

**Accès toujours possible aux détails complets :** Les informations complètes de l'extension — y compris le statut, le canal, la date de publication et les exigences de version — sont toujours disponibles en déplaçant le focus vers la section **« Autres détails »** ou en appuyant sur le bouton approprié dans la boîte de dialogue de la boutique d'extensions. addonStoreLite optimise uniquement l'annonce vocale ; il ne cache jamais de données.
