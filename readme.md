## Description
**addonStoreLite** is a smart, lightweight add‑on that transforms how NVDA announces items in the Add‑on Store. It strips away unnecessary clutter—like status labels, email addresses, and extra version numbers—and presents only the essential information: **add‑on name, version, author, install date, and a clear “Disable” indicator for disabled add‑ons**. The result? A clean, concise announcement that lets you scan through available add‑ons effortlessly.

> **Works automatically:** Just install the add‑on and focus any list item in the Add‑on Store. NVDA will immediately speak the streamlined version. No configuration needed.

<br>

## Why You’ll Love It
Browsing the Add‑on Store used to mean listening to long, repetitive strings of text—status, channel, publisher, and more. With **addonStoreLite**, you hear only what matters, saving time and reducing cognitive load. It intelligently adapts to both the classic labeled format (NVDA 2025.x) and the new positional format introduced in NVDA 2026.x, ensuring flawless performance across all tabs:

* ✅ Installed add‑ons
* ✅ Available add‑ons
* ✅ Updates tab
* ✅ Beta & Dev channels

## What It Removes
* Status labels like “Status: Enabled” (only “Disable” is spoken when applicable)
* Email addresses inside angle brackets
* Channel names (External, Stable, Beta, Dev)
* Redundant version fields (e.g., “Minimum NVDA version”, “Last tested”)
* Punctuation and extra spaces for a natural flow

**Still have access to full details:** The complete add‑on information—including status, channel, publication date, and version requirements—is always available by moving focus to the **"Other details"** section or by pressing the appropriate button in the Add‑on Store dialog. addonStoreLite only streamlines the spoken announcement; it never hides any data.
